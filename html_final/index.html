Analyzing Traffic Crash Data

Problem:

Can we use Machine Learning algorithms to predict if certain environmental conditions contribute to fatal accidents leveraging a traffic crash dataset from 2009-2017?

Findings:

Our initial thought and hope was that we would be able to find a correlation between crashes, crashes with fatal or non-fatal injuries, and various road, weather, and day conditions. However, after working through several regression models, we came to the conclusion that there is not measurable correlation between crashes, crashes with fatal or non-fatal injuries, and other various conditions. Given the dataset we leveraged, traffic crashes appear to be random or atleast connected to variables not included in our dataset.

If we had more time for a deeper analysis, we would try to leverage the Stochastic Gradient Descent (SGD) model. Since our dataset was relatively large, this model would help us better programmatically identify correlations between different variables, ones that we might not have thought would make an impact.
